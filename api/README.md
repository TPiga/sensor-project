## Description

[Koa](https://github.com/koajs/koa) framework TypeScript starter repository.

## Installation

```bash
$ yarn
```

## Running the app

```bash
# start database
$ docker-compose up -d

# watch mode
$ yarn start:dev
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov
```
